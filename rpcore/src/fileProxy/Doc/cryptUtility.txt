Instructions for building and using cryptUtility.exe
====================================================


Change directory to Code/cryptUtility.  
Edit cryptUtility.mak to set up the object file directory for 
cryptUtility as indicated in DirectoryStructure.txt.  Then
compile the cryptUtility:
    make -f cryptUtility.mak
 
options for cryptUtility
    cryptUtility.exe -help
    cryptUtility.exe -GenKey keytype outputfile
    cryptUtility.exe -Sign keyfile algorithm inputfile outputfile
    cryptUtility.exe -Verify keyfile inputfile
    cryptUtility.exe -Canonical inputfile outputfile
    cryptUtility.exe -RSATest keyfile
    cryptUtility.exe -PolicyCert certfile outputfile programname
    cryptUtility.exe -Encrypt keyfile inputfile outputfile
    cryptUtility.exe -Decrypt keyfile inputfile outputfile
    cryptUtility.exe -HexquoteTest
    cryptUtility.exe -SignHexModulus keyfile input-file output-file
    cryptUtility.exe -HashFile input-file [alg]
    cryptUtility.exe -makePolicyKeyFile input-file outputfile
    cryptUtility.exe -makeServiceHashFile input-file outputfile
    cryptUtility.exe -VerifyQuote xml-Quote xml-aik-cert
